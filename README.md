# Homebrew Tap for atra

Official Homebrew tap for atra — [Atrahasis](https://atrahasis.dev) CLI.

## Quick Install

```bash
brew install atrahasisdev/tap/atra
```

## Or Step by Step

```bash
brew tap atrahasisdev/tap
brew install atra
```

## Upgrade

```bash
brew upgrade atra
```

## Usage

```bash
atra GET https://api.example.com/users
atra POST https://api.example.com/users name:John age:25
atra GET https://api.example.com/auth -u user:pass
atra GET https://api.example.com -a "status eq 200"
```
